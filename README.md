# Student Record Management System

## 📌 Project Overview
The Student Record Management System is a console-based Java application that allows users to manage student records efficiently. The project implements CRUD (Create, Read, Update, Delete) operations using ArrayList and demonstrates core Object-Oriented Programming (OOP) concepts.

## 🚀 Features
- Add Student Records
- View All Student Records
- Update Student Details
- Delete Student Records
- Menu-Driven Interface
- Uses ArrayList for dynamic data storage

## 🛠 Technologies Used
- Java
- ArrayList
- Object-Oriented Programming (OOP)
- VS Code 

## 📚 Concepts Covered
- Classes and Objects
- Constructors
- Encapsulation
- Getters and Setters
- ArrayList Collection
- Loops
- Conditional Statements
- Method Overriding (`toString()`)



## 🎯 Learning Outcomes
This project helps in understanding:
- Java Fundamentals
- Collection Framework (ArrayList)
- CRUD Operations
- Data Management Using Objects
- Encapsulation and OOP Principles
